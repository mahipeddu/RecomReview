┌─ ACCESS THE APP ───────────────────────────────────────────────────────────┐
│                                                                              │
│  Local URL:    http://localhost:8501                                         │
│  Network URL:  (shown in terminal output)                                    │
│                                                                              │
│  Browser:      Any modern browser (Chrome, Firefox, Safari, Edge)           │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ USAGE WORKFLOW ───────────────────────────────────────────────────────────┐
│                                                                              │
│  1. Upload PDF     → Click or drag & drop your research paper               │
│  2. Wait           → 5-10 seconds for processing                             │
│  3. View Results   → Top N reviewers with similarity scores                  │
│  4. Explore        → Interactive charts and detailed statistics              │
│  5. Download       → Export results as CSV                                   │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ FEATURES ─────────────────────────────────────────────────────────────────┐
│                                                                              │
│  ✓ PDF Upload              → Drag-and-drop or click                         │
│  ✓ AI Analysis             → Pretrained embeddings (all-mpnet-base-v2)      │
│  ✓ Top N Reviewers         → Customizable (5-20)                            │
│  ✓ Similarity Scores       → 0.0 to 1.0 (higher is better)                  │
│  ✓ Interactive Charts      → Plotly visualizations                          │
│  ✓ Detailed Statistics     → Per-reviewer metrics                           │
│  ✓ CSV Export              → Download results                               │
│  ✓ GPU Support             → Automatic detection                            │
│  ✓ Real-time Processing    → Live updates                                   │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ SIDEBAR SETTINGS ─────────────────────────────────────────────────────────┐
│                                                                              │
│  • Number of recommendations  (5-20)                                         │
│  • Show detailed statistics   (on/off)                                       │
│  • Show visualization         (on/off)                                       │
│  • Model status              (loaded/error)                                  │
│  • Device info               (GPU/CPU)                                       │
│  • Database statistics       (authors, papers)                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ KEYBOARD SHORTCUTS ───────────────────────────────────────────────────────┐
│                                                                              │
│  R              → Rerun the app                                              │
│  C              → Clear cache                                                │
│  Ctrl+C         → Stop server (in terminal)                                  │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ RESULTS EXPLANATION ──────────────────────────────────────────────────────┐
│                                                                              │
│  Rank         → Position in top N                                            │
│  Author       → Reviewer name                                                │
│  Max Score    → Highest similarity score (0.0-1.0)                           │
│  Papers       → Number of papers in database                                 │
│  Avg Score    → Average across all papers                                    │
│                                                                              │
│  Perfect Match (★) → Score ≥ 0.999 (paper likely in database)               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ COMMON ISSUES & SOLUTIONS ────────────────────────────────────────────────┐
│                                                                              │
│  ❌ "Embeddings cache not found"                                             │
│     → Generate cache: cd pretrained_embedding && python3 semantic_*.py       │
│                                                                              │
│  ❌ "Failed to extract text"                                                 │
│     → PDF might be scanned (image-based), try different PDF                  │
│                                                                              │
│  ❌ Port already in use                                                      │
│     → Use different port: --server.port 8080                                 │
│                                                                              │
│  ⚠️ Slow performance                                                         │
│     → Check if GPU is being used (shown in sidebar)                          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
